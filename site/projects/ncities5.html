Site is made using a static site generator I am writing, which uses the mustache templating engine and a big YAML file with all the posts and links and sections and this very text I'm writing rn!

A python script goes through a source directory which contains a) this YAML file, and b) the templates and then provides the dict that it gets from reading the YAML file to the templates which do all the rest. Also does stuff to make a seperate file for each project and article and so on. It then puts all the compiled HTML into another folder along with CSS ive written and the fonts and images and so on, and then another program I've written to interface with the neocities API uploads all the files, and syncs the local folder to the server one.
